# Adaptive-Math-Misconception-Tutor

This project is an AI-powered tutoring system designed for below-average middle school students. Its core purpose is not just to mark answers as right or wrong, but to diagnose *why* a student is wrong, model their understanding over time, and adapt instruction accordingly.

The system encodes the judgment of a human tutor into software by treating student mistakes as data and learning signals. At a high level, the project combines rule-based math reasoning, lightweight statistical modeling, and generative AI for explanation. 

It is intentionally built as a real-world system: one that interacts with imperfect humans, updates beliefs under uncertainty, and produces clear, encouraging feedback. This makes it both an educational tool and a strong demonstration of applied AI system design.

How to Run: 

Execute from the command line: python3 phase_2.py
